# YDM Site Audit — 2026-05-04

Read-only audit run against `localhost:3000` (Next.js dev). Fetched every route in `archive/wordpress/scrape/route-map.json`, ran HTML-level health checks, categorized findings.

## Method

- 64 routes in `route-map.json` → 46 unique base paths after stripping query strings (per spec: "use the base path only" for `index_filter` routes — the 18 `?author=`, `?category=`, `?tag=` variants all render their base index in Next.js routing).
- All 46 unique paths fetched via `curl -sL` against the running dev server.
- Each saved HTML inspected for: HTTP status, title content, body text length, `[MISSING:` markers, `<img src="">`, `<a href="">`, `<a href="#">`, `theme-dev.cmsmasters` / `faith-connect.cmsmasters` host references, `lorem ipsum`, and the legacy `– YDM – Yeshua Deliverance Ministries` title suffix.

## CONTENT EDITS NEEDED (admin / DB row updates)

### A. Title-suffix bloat — 33 routes

Every route below has its `meta.title` row in `page_content` ending with the legacy WP suffix `– YDM – Yeshua Deliverance Ministries`, often *combined* with a duplicate `| Yeshua Deliverance Ministries` from the metadata layer. Result: the browser tab shows the site name two or three times.

The CHANGELOG records `fix(L cleanup): strip "– YDM – Yeshua Deliverance Ministries" suffix from sermon + ministry meta.titles` — that pass only covered `sermons.*` and `ministries.*` page_keys. Every other type still has the suffix in DB.

Affected:
- `/` — `YDM – Yeshua Deliverance Ministries – Ministry by Bishop H.O. Wilson`
- `/about`, `/ask`, `/blog`, `/contact`, `/events`, `/gallery`, `/give`, `/guestbook`, `/live`, `/ministries`, `/prayer`, `/sermons`, `/team`
- All 9 blog detail pages
- Both events: `/events/weekly-bible-study-group`, `/events/young-adults-connect`
- Both give campaigns: `/give/supportydm`, `/give/techfund`
- Both locations: `/locations/maltoncog`, `/locations/westtoronto`
- All 4 team pages: `/team/bishop-huel-wilson`, `/team/bishopwilson`, `/team/clementinawilson`, `/team/huel-and-clementina-wilson`

**Fix shape:** one-shot SQL update — strip ` – YDM – Yeshua Deliverance Ministries` (and trailing ` | Yeshua Deliverance Ministries`) from every `meta.title` row. Pattern is similar to the seed-fixups script's existing typo replacements. ~10 lines of TS.

### B. Placeholder `href="#"` anchors — 6 routes

WP-scraped `body_html` still contains anchor tags with `href="#"` (likely social-share widgets, "expand" toggles, or accordion triggers from the CMSMasters theme). They render as live links that go nowhere.

- `/blog/experiencing-the-freedom-of-gods-presence` — 7 hash-hrefs
- `/blog/four-things-our-church-should-still-focus-on` — 4 hash-hrefs + 6 `href=""`
- `/blog/how-do-i-intercede-for-family-who-dont-believe-in-jesus` — 5 hash-hrefs
- `/blog/your-mission-should-you-choose-to-accept-it` — 7 hash-hrefs
- `/locations/maltoncog` — 5 hash-hrefs
- `/locations/westtoronto` — 5 hash-hrefs

**Fix shape:** body_html sanitizer pass (similar to the existing `sermon-body` and `ministry-body` strip passes referenced in CHANGELOG). Could either strip the anchors entirely, or unwrap them to keep the inner text. ~20 lines of TS.
